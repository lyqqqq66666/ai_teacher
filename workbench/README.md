# 希沃智教π · 作业批改系统

## 主流程

1. **创建批改项**：选择学科与模式，拍照/上传题目；AI 生成判题标准  
   - **模式 A**：题目 + 答案册  
   - **模式 B**：只导题目 → AI 出参考答案与过程标准  
2. **确认批改任务**：老师确认后锁定标准，生成一次作业批改任务  
3. **批量上传作业**：上传同学作业图 → 自动归属学生 → 呈现批改效果  

## 本地启动

```bash
cd demo
python3 -m http.server 5173
```

打开 http://127.0.0.1:5173

> 需用本地服务器打开，以便样例图片 `fetch` 导入。

## 最快体验

- 工作台 → **一键演示 · 数学模式 A / B**  
- 或侧栏 **导入一次批改** 三步走  
- 第 3 步可用「一键导入本学科样例作业」或自己上传图片  

## 文件结构

```
demo/
  index.html
  css/app.css
  js/app.js          # 界面与流程
  js/grader.js       # 批改引擎（演示级）
  data/demo-data.js  # 题库 / 学科 / 样例元数据
  public/thumbs/    # 数学17 + 语文15 + 奥数5
```

任务保存在浏览器 `localStorage`（键：`seewo_pi_tasks_v1`）。
